# 1000yo

A web app implementation scaffold for *Thousand Year Old Vampire*.

## Project status

This repository is initialized as a browser-based JavaScript app with:

- A minimal memory-journal UI (`index.html` + `src/main.js`)
- Core `Character` game logic in JavaScript
- Node built-in unit tests for core behavior

## Quickstart

```bash
npm test
npm run dev
```

Then open: `http://localhost:4173`
